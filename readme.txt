## Setup Instructions
To set up the SDA-PHASE-3 project on your local machine, follow these steps:
1. **Clone the repository**:
   ```bash
   git clone https://github.com/Ambritzk/SDA-PHASE-3.git
   ```
2. **Navigate into the project directory**:
   ```bash

   ```
3. **Install dependencies**:
   ```bash
   npm install
   ```

4. **Run the project**:
   ```bash
   npm start
   ```
